# P2P Gaming Platform

Build peer-to-peer on-chain gaming platforms on Monad with fair game logic, matchmaking, wager escrow, and real-time multiplayer.

## What This Skill Does

Equips your agent with everything needed for on-chain gaming — from Solidity game contracts with commit-reveal fairness to WebSocket real-time updates and wager escrow systems.

## Skills Covered

- Turn-based and real-time game logic in Solidity
- Commit-reveal schemes and VRF randomness
- Matchmaking and ELO rating systems
- Escrow-based wager contracts
- WebSocket real-time game state
- Leaderboards and player profiles

## Ideal For

Builders creating multiplayer on-chain games, P2P betting platforms, or tournament systems on Monad.

## Installation

```
cp -r p2p-gaming/ your-project/.claude/skills/
```
