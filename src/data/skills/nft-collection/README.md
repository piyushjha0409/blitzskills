# NFT Collection Launcher

Launch NFT collections on EVM chains with ERC-721/1155, generative art, IPFS metadata, and complete minting mechanics.

## What This Skill Does

Equips your agent with everything for NFT launches — from writing ERC-721/1155 contracts with allowlist minting to setting up IPFS metadata, reveal mechanics, and building mint page UIs.

## Skills Covered

- ERC-721 and ERC-1155 token contracts
- Allowlist, Dutch auction, and lazy minting
- IPFS metadata and reveal mechanics
- Generative art with trait randomization
- ERC-2981 royalty standard
- Mint page UI with wallet connection

## Ideal For

Builders launching NFT collections, PFP projects, or any NFT-based application on EVM chains.

## Installation

```
cp -r nft-collection/ your-project/.claude/skills/
```
