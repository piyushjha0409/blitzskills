# Token Deployer

Deploy and manage ERC-20 tokens on EVM chains with supply management, vesting, airdrops, and governance.

## What This Skill Does

Equips your agent with token deployment expertise — from creating ERC-20 contracts with mint/burn mechanics to building Merkle-based airdrop systems, vesting schedules, and governance with voting.

## Skills Covered

- ERC-20 token creation with extensions
- Mintable, burnable, pausable patterns
- Merkle tree airdrop contracts
- Linear and cliff vesting schedules
- Governor + Timelock governance
- DEX liquidity pool setup

## Ideal For

Builders creating project tokens, governance systems, airdrop campaigns, or any token-based application on EVM chains.

## Installation

```
cp -r token-deployer/ your-project/.claude/skills/
```
