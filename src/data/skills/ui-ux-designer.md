## About

Design-focused template for hackathon participants who bridge the gap between aesthetics and usability. Ideal for designer-developers who code their own designs.

## Skills Breakdown

- **UI/UX Design** — End-to-end product design from user research to high-fidelity prototypes in Figma
- **Design systems** — Component libraries, design tokens, and scalable UI patterns
- **Motion & interaction** — Framer Motion, Lottie, CSS animations that make interfaces feel alive
- **Frontend implementation** — React + Tailwind to turn designs into pixel-perfect, responsive UIs

## Ideal For

Designers who ship code — turning wireframes into production-ready interfaces with thoughtful micro-interactions and polished visual details.
