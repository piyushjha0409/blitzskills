# Prediction Market Builder

Build on-chain prediction market protocols on EVM chains with conditional tokens, automated market makers, and oracle-based resolution.

## What This Skill Does

Equips your agent with the knowledge to build complete prediction market systems — from creating markets with conditional tokens to integrating oracles for resolution and building liquidity pools with scoring rules like CPMM and LMSR.

## Skills Covered

- Conditional token standards (ERC-1155)
- CPMM and LMSR scoring rules
- Oracle integration (Chainlink, Pyth, UMA)
- Automated settlement and fee distribution
- Liquidity pool design and incentives
- Market browsing and position management UI

## Ideal For

Builders creating prediction markets, binary options platforms, or event-based trading protocols on EVM chains.

## Installation

Copy this folder into your project's skills directory:

```
cp -r prediction-market/ your-project/.claude/skills/
```
